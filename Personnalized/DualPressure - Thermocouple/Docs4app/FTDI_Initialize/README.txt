First, you'll need to install D2XX Drivers :
https://ftdichip.com/drivers/d2xx-drivers/

For Python application with FTDI chip, let's install the ftd2xx python library :
pip install ftd2xx

This will allow you to use directly the D2XX Drivers through ftd2xx python library.
(You can also use the pyftdi OR pylibftdi python libraries)


For more infos, you may have a look to the link below :
https://iosoft.blog/2018/12/02/ftdi-python-part-1/
